## Background Options

### Solid Colors
- Choose from curated preset colors
- Use the color picker for any custom color
- Enter specific hex codes (e.g., #2c2419)

### Gradients
- Pre-made gradient combinations
- Smooth transitions between two colors
- Various directions and angles

### Image Backgrounds
- Upload any image as a full background

## Font Controls

### Available Font Families

**Bengali (Bangla) Fonts:**
- **Noto Serif Bengali** — Elegant serif style
- **Hind Siliguri** — Clean sans-serif
- **Tiro Bangla** — Traditional look

**English Fonts:**
- Various professional options available depending on the card type

### Font Properties You Can Adjust

| Property | What It Does | Range |
|---|---|---|
| **Size** | How large the text appears | Varies by element (10px-108px) |
| **Weight** | How thick/bold the letters are | Normal → Extra Bold |
| **Color** | The color of the text | Any color |
| **Alignment** | Text direction | Left, Center, Right |
| **Letter Spacing** | Space between individual letters | Custom values |

### Text Effects

**Text Shadow**
Add depth to your text with shadow presets:
- **None** — No shadow
- **Soft** — Subtle, blurred shadow
- **Hard** — Sharp, defined shadow
- **Glow** — Bright glow effect
- **Outline** — Shadow that creates an outline look
- **Custom Angle** — Control the shadow direction

**Text Stroke**
Add an outline around your text:
- **Width** — How thick the outline is (0-5px)
- **Color** — What color the outline is

## Visibility Controls

Every card type has visibility toggles for individual elements. Common toggleable items:
- ✅ Logo
- ✅ Day name / Week
- ✅ Date
- ✅ Headline / Title
- ✅ QR Code
- ✅ Ad Banner
- ✅ Social Media handles

Simply flip the toggle to show or hide each element.

---

## Frame / Border Options

Add a border around your card:
- **Enable/Disable** — Toggle the frame on or off
- **Color** — Choose any border color
- **Thickness** — From thin to thick borders

---

## Ad Banner

Some card types support an **Ad Banner** area for advertising or promotions:
- Upload a banner image
- **Zoom** — Control how zoomed in the banner is
- **Position** — Adjust horizontal (X) and vertical (Y) position

---

## Element Layout (Drag Mode)

For cards that support it:
1. Enter **Drag Mode** from the toolbar
2. Elements become draggable
3. Position elements in different slots:
   - Top Left, Top Right
   - Bottom Left, Bottom Right
   - Center (some themes)
4. Exit drag mode when done

### Available Draggable Elements
Depending on the card type:
- Logo
- Date/Week
- Social Media
- Website
- CTA (Call to Action)
- Favicon

## Color Tips

1. **Use brand colors** — Consistency builds recognition
2. **Dark backgrounds with light text** — Maximum readability
3. **Avoid pure black (#000000)** — Use dark grays (#1a1a1a or #2c2419) for a warmer feel
4. **Test on mobile** — Colors can look different on phone screens
5. **Use the hex input** — For exact brand color matching, type the exact hex code
