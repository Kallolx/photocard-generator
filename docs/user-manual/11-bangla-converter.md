# Bangla Converter

## Professional Bengali Text Conversion Tool

The **Bangla Converter** is a specialized utility for anyone who works with Bengali (Bangla) text. It helps you convert text between different Bengali encoding formats and transliterate English text into Bangla.

---

## What Is It For?

In the Bengali-speaking world, there are different ways text is encoded:
- **Unicode** — The modern, standard format used by websites and social media
- **Bijoy** — A legacy format used in many print publications, newspapers, and older software

The Bangla Converter helps you move between these formats seamlessly.

---

- From the sidebar: **Bangla Converter**
- Or from the Dashboard tools section

## Features

### 1. English to Bangla (Phonetic)
- Type English letters and they're automatically converted to Bengali script
- Uses phonetic transliteration — type how it sounds
- Example: typing "ami" produces "আমি"

### 2. Bijoy to Unicode
- Paste text written in Bijoy encoding
- Get clean Unicode output
- Essential for converting content from print newspapers or older documents

### 3. Unicode to Bijoy
- Paste modern Unicode Bengali text
- Get Bijoy-encoded output
- Useful when you need to work with legacy publishing systems

### 4. One-Touch Swap
- Instantly swap your input and output text
- Useful for quickly checking conversions both ways

### 5. Character Counter
- See the character count of your text
- Helps ensure your content fits within social media character limits

## When Would You Use This?

| Scenario | Use Case |
|---|---|

| You need to send content to a legacy publishing system | Convert Unicode → Bijoy |

| You're checking social media character limits | Use the character counter |
